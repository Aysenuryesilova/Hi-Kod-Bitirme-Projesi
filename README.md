# 🌿 Carbon Footprint Calculator

This is my first web development project, built as part of the Hi-Kod training program. It's a simple but functional tool that helps people calculate their personal carbon footprint based on daily habits.

## About This Project

The carbon footprint represents the total amount of greenhouse gases produced by an individual's activities. This calculator allows users to estimate their impact by answering a few simple questions about their lifestyle.

**What it does:**
- Calculates carbon footprint based on car usage, air travel, meat consumption, and electricity use
- Shows results with color-coded indicators (green, yellow, red)
- Gives personalized tips to reduce environmental impact
- Saves calculation history and displays it on a chart
- Allows sharing results on social media

## Technologies I Used

| Technology | What I Used It For |
|------------|-------------------|
| **HTML5** | Page structure and content |
| **CSS3** | Design, colors, layout, responsive styling |
| **JavaScript** | Calculations, history management, interactions |
| **Chart.js** | Displaying calculation history as a chart |
| **QRCode.js** | Generating QR codes for mobile access |
| **Font Awesome** | Icons and visual elements |
| **LocalStorage** | Saving user calculations in the browser |

## How to Run

1. Download or copy the `index.html` file
2. Open it in any modern web browser
3. Start calculating!

No server or installation required. It works directly in your browser.

## Key Features

- **Carbon Calculator:** Four categories (car, flight, meat, electricity) with sliders
- **Color Indicators:** Low (green), medium (yellow), high (red) results
- **Personal Tips:** Customized suggestions based on your result
- **History Log:** All calculations saved automatically
- **Chart View:** See your progress over time
- **Share Options:** X, LinkedIn, WhatsApp, and copy to clipboard
- **QR Code:** Quick mobile access
- **Language Options:** Turkish, English, German, French
- **Fully Responsive:** Works on phone, tablet, and desktop
